# Housing Market Trends

## About Project
This project analyzes housing market data to study how different factors affect house prices. It helps understand trends in the real estate market using data analysis and visualization.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset contains:
- Income
- House Age
- Number of Rooms
- Population
- House Price

## Features
- Load housing dataset
- Analyze house price trends
- Find average, highest, and lowest prices
- Visualize data using graphs

## Output
- Dataset preview  
- House price analysis  
- Graph showing housing market trends  

## Conclusion
This project helps understand how factors like income and house details influence house prices.
